Homeland

Homeland is is an exquisite and exquisite open source community forum system. Its features are derived from accumulated over the years of Ruby China, small and practical.

The original project was founded in 2009, and kept the continuous function conversion under the practice scenario of [Ruby China](https://ruby-china.org, thus reaching the current state.

Homeland can help you quickly build a forum system, you only need a few steps to install the script. At the same time, this project will continue to be maintained and developed with the development of Ruby China.
